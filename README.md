# 🛡️ SendShield

SendShield is a Chrome extension that helps protect crypto users from sending funds to risky wallet addresses.

## Features

* Scam Wallet Detection
* Sanctioned Wallet Detection
* High-Risk Wallet Detection
* Real-Time Warning Popups
* External Threat Intelligence Database (`wallets.json`)
* Works on Crypto Exchange Withdrawal Pages
